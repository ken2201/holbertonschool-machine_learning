# Linear Algebra

This directory contains solutions for the Linear Algebra project, which is the first project of the first trimester in the Holberton School Machine Learning curriculum.

## Learning Objectives

- What is a vector?
- What is a matrix?
- What is a transpose?
- What is the shape of a matrix?
- What is an axis?
- What is a slice?
- How do you slice a vector/matrix?
- What are element-wise operations?
- How do you concatenate vectors/matrices?
- What is the dot product?
- What is matrix multiplication?
- What is Numpy?
- What is parallelization and why is it important?
- What is broadcasting?

## Files

| File | Description |
|------|-------------|
| `0-slice_me_up.py` | Slicing arrays |
| `1-trim_me_down.py` | Trimming a matrix |
| `2-size_me_please.py` | Get the shape of a matrix |
| `3-flip_me_over.py` | Transpose a matrix |
| `4-line_up.py` | Add two arrays element-wise |
| `5-across_the_plane.py` | Add two matrices element-wise |
| `6-howdy_partner.py` | Concatenate two arrays |
| `7-gettin_cozy.py` | Concatenate two 2D matrices along a specific axis |
| `8-ridin_bareback.py` | Perform matrix multiplication |
| `9-let_the_butcher_slice_it.py` | Slice various matrices |
| `10-ill_use_my_scale.py` | Calculate the shape of a numpy.ndarray |
| `11-the_western_exchange.py` | Transpose a numpy.ndarray |
| `12-bracin_the_elements.py` | Element-wise operations on numpy.ndarrays |
| `13-cats_got_your_tongue.py` | Concatenate two numpy.ndarrays |
| `14-saddle_up.py` | Perform matrix multiplication using numpy |

## Requirements

- Python 3.5+
- Numpy 1.15+
- All files should end with a new line
- The first line of all files should be `#!/usr/bin/env python3`
- All modules should have documentation
- All classes should have documentation
- All functions should have documentation
- Code should use pycodestyle (version 2.5)
